# 아기티큐 200티큐 자동화 스크립트

> **안드로이드 + Termux + 텔레그램 푸시 알림 기반**  

## 특징
- 메인티커 TQQQ외 타 티커 사용가능
- 아기티큐 오리지널 타 티커 기준점도 TQQQ로 잡는건 setup_final_tqqq200이며, setup_final은 메인티커 변경시 해당티커의 200일선으로 변경됨

## 설치 및 사용법

1. 텔레그램 봇 API 및 ID 확인
2. **Termux, Termux:boot 설치** (F-Droid)

이후 자세한 안내는 사용가이드.txt 참조

## 다운로드

우측 상단 "Code" → "Download ZIP“

##  중요 주의사항
- 텔레그램 봇 키 및 채팅 ID는 개인정보입니다. 철저히 관리하세요.
- 본 스크립트는 정보 제공용 참고자료입니다.
- 투자 권유나 자문이 아닙니다.
- 모든 투자 결정과 책임은 본인에게 있습니다.
- 작성자는 어떠한 투자 결과에 대해서도 법적 책임을 지지 않습니다.
